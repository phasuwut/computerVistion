Set Properties for Webcam Acquisition on Linux

R2020b

New Features
expand all

Live Editor Task: Acquire images from USB webcam interactively and generate MATLAB code in a live script.

Use the new Live Editor Task, Acquire Webcam Image, to connect to a webcam, set properties, and capture images without writing MATLAB code. The task gives you controls that help you to set values of webcam resolution and other image-specific and device-specific properties. The task also automatically generates MATLAB code that becomes part of your live script.

To use tasks in the Live Editor, on the Live Editor tab, in the Task menu, select a task. Alternatively, in a code block in a live script, begin typing the task name and select the task from the suggested command completions. For more information about Live Editor tasks generally, see Add Interactive Tasks to a Live Script (MATLAB).

Create multiple connections to the same webcam

The MATLAB Support Package for USB Webcams now enables you to create multiple connections to the same webcam. If a connection to a webcam already exists in the workspace, MATLAB returns the existing webcam connection. Prior to R2020b, creating multiple connections to the same webcam resulted in an error.

Fixes to webcam Object for Linux distributions containing GStreamer version 1.14 and above (January 2021, Version 20.2.1)

Creating a webcam object for cameras that support only compressed video formats failed in Linux distributions containing GStreamer version 1.14 and above. This issue has been fixed now.

Support for USB webcams with NV12 video pixel format

The MATLAB Support Package for USB Webcams now enables you to connect to USB webcams that support only NV12 video pixel format.

snapshot Function : Improved performance when capturing images

The snapshot function shows improved performance when capturing images from a USB Webcam. The function is up to 6x faster with some variation in its performance based on the quality of your internet connection.

For example, this code is about 6x faster than in the previous release.

function data = timingTest(camObj) 
    data = cell(100,1); 
    for cnt = 1:100 
     data{cnt} = snapshot(camObj); 
    end 
 end

R2020a: 10.01s

R2019b: 58.37s

The code was timed on Google Chrome (Version 79.0.3945.117) with a 640x480 webcam using the timeit(@timingTest) function.

R2019b

Bug Fixes
expand all

Updates to webcamlist Function on Ubuntu 18.04 (October 2019, Version 19.2.1)

For the MATLAB Support Package for USB Webcams installed on Ubuntu® 18.04, the webcamlist function did not return the list of available UVC-compliant webcams as the output. This issue is now resolved and the webcamlist function returns the list of all UVC-compliant cameras connected to the system.

Fixes to webcam Object on macOS Mojave (November 2019, Version 19.2.2)

Creating a webcam object using the MATLAB Support Package for USB Webcams installed on macOS Mojave caused MATLAB to crash. This issue has been fixed. To use the webcam, you must allow camera access to MATLAB when prompted.

Updates to preview function (October 2020, Version 19.2.3)

Using the preview function in parallel with a MATLAB function callback caused the MATLAB window to freeze. This issue has been fixed.

MATLAB Online: Acquire live images from USB webcams in MATLAB Online

You can now connect to and stream images from your USB webcam in MATLAB Online™. Connect a USB webcam to your computer and give your browser access to your camera. All the functions in the MATLAB Support Package for USB Webcams are available for MATLAB Online.
