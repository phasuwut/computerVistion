Example Not Found

MATLAB did not find the example that you specified.

To solve this issue:

  • Enter the demo function with a different argument.
  • Update Help browser preferences to display examples for all installed products.

To update the Help browser preferences:

  1. In the Help browser, click the cog icon.
  2. In the Preferences dialog box, under MATLAB, select the Help pane.
  3. For Selected Products, click Select All.
  4. Click OK.
